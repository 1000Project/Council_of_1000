# 6. Targeting Rules

# 7. Replacement and Prevention Effects

If two replacement effects would apply simultaneously:

- the **controlling player** chooses the order  
- after one applies, the other evaluates if it still applies  

Prevention effects negate damage or events before they occur.

# 8. Game State Priority

Players alternate priority to take actions.

Priority passes when:

- both players decline to act  
- a stack item resolves  
- a phase ends  

The active player always receives priority first.

# 9. Illegal Actions

If an illegal action is attempted:

- the action is canceled  
- no costs are paid  
- the game state rewinds to the last legal point  

Tournament play uses judge intervention.
