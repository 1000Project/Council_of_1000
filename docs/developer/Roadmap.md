# Development Roadmap  
_The Path From Concept to Launch for the Council of 1000 TCG_

The Council of 1000 TCG is a multi-phase project designed for long-term growth, technical stability, and a polished player experience. This roadmap outlines the complete development lifecycle from pre-production to launch and post-launch live service.

Each phase builds upon the previous, ensuring predictable progress while allowing creative and technical flexibility.

- Core gameplay principles (lanes, influence, resolve, turn structure)  
- High-level card type definitions

- Core Mechanics Overview

- Expanded rules framework
